# Getting Started
Create the project skeleton using [Spring Initializr](https://start.spring.io)

Add depencencies:

- Web
- JPA
- H2
- Lombok


**@SpringBootApplication** is a meta-annotation that pulls in component scanning, autoconfiguration, and property support.
